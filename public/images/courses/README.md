# Course Images

This directory contains course thumbnail and hero images.

## Image Requirements

- **Format**: JPG or PNG
- **Recommended dimensions**:
  - Course card thumbnails: 1280x720px (16:9 aspect ratio)
  - Course hero images: 1280x720px (16:9 aspect ratio)
- **File size**: Optimize images to be under 500KB for better performance
- **Naming convention**: Use the course slug as the filename

## Image Placeholders

Until actual images are added, the components will gracefully handle missing images by:
- Showing a gradient background fallback
- Not breaking the layout

## Tips

- Use professional, relevant images that represent the course content
- Ensure images are high quality but optimized for web
- Consider using stock photos or custom graphics
- Maintain consistent styling across all course images
