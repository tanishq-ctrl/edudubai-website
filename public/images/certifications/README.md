# Certification Images

This directory contains certification thumbnail images for the Partner Certifications section.

## Image Requirements

- **Format**: JPG or PNG
- **Recommended dimensions**: 1280x720px (16:9 aspect ratio)
- **File size**: Optimize images to be under 500KB for better performance
- **Naming convention**: Use the certification ID as the filename

## Required Images

Add the following images to this directory:

1. `regulatory-compliance-specialist.jpg` - Regulatory Compliance Specialist (RCS)
2. `know-your-customer-specialist.jpg` - Know Your Customer Specialist
3. `certified-compliance-manager.jpg` - Certified Compliance Manager (CCM)

## Image Placeholders

Until actual images are added, the components will gracefully handle missing images by:
- Showing a gradient background fallback
- Not breaking the layout

## Tips

- Use professional, relevant images that represent the certification content
- Ensure images are high quality but optimized for web
- Consider using stock photos or custom graphics
- Maintain consistent styling across all certification images
