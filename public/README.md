# Public Assets

## Logo

Please add your EduDubai logo image here with the filename:

**`edudubai-logo.png`**

### Requirements:
- File name: `edudubai-logo.png` (or `.svg` if you have an SVG version)
- Recommended format: PNG with transparent background
- Recommended dimensions: At least 360x120px (2:1 aspect ratio) for best quality
- The logo will be displayed at:
  - Header: 180px width (height auto, max 48px)
  - Footer: 150px width (height auto, max 32px)

### Notes:
- The logo should have a transparent background to work on both light and dark backgrounds
- Next.js Image component will automatically optimize the image
- The logo is set to `priority` loading in the header for faster initial load
